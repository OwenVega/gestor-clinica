# 🏥 Gestor de Citas Médicas

Plataforma diseñada para optimizar el flujo de atención, administración de pacientes y agendamiento de citas en un entorno clínico.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Java
* **Gestor de dependencias:** Maven
* **Base de Datos:** SQL Server

## 💻 Sobre el Repositorio
Este repositorio contiene el código fuente backend del sistema de gestión, incluyendo la estructura de las clases principales, la conexión a la base de datos relacional y la lógica de negocio para la administración clínica.
